### 1. 伪类
``` python
给某个标签添加特殊的效果(样式)
a标签四个伪类：
    :link 未访问
    :visited 访问过
    :hover 鼠标悬停
    :active 鼠标按下
    
我们可以用span标签区分样式    
```

### 3.标签：
``` python
(1)h1-h6标签：
    h1 logo整个网页只有一个h1
    h2 副标题(预留项)
    h3 模块标题
    h4 模块的子标题
    h5/h6 一般不用
(2) p 段落
    strong 强调粗体
    em 强调斜体
    span 区分样式
    ul/ol 无序列表/有序列表
    li 列表项
    dl 自定义列表
    dt 自定义列表的标题
    dd 自定义列表列表项
    mark 标记
(3)img： 
    1)当修改宽/高任意一个时，图片会等比例缩放
    2)title鼠标悬停时现实的内容，绝大多数标签都能添加title属性
    3)alt图片加载失败显示的内容，图片加载失败没有alt时，显示title内容
    
```
