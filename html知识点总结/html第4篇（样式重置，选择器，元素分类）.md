### 1. 默认样式
``` python
(1) 找出有margin默认样式的元素？
    body,h1`h6，hr,p,dl,dd
(2) 找出有默认padding默认样式的元素？
    无
(3) 找出同时有有margin,padding默认样式元素？
    ul，ol
    
    reset样式：
    单凡是浏览器默认的样式，都不要使用，并且有针对性的进行重置，只重置在页面中出现过的标签
```
### 2.选择器
``` python
(1) 群组选择器：用于不同的元素执行相同的代码片段
(2) id选择器：id选择器唯一，配合js使用
(3)类选择器：比较灵活，用于页面多个同样样式的元素
(4)标签选择器：由于标签选择器查找范围过大，在css reset中使用
(5)包含选择器：通过制定一个父级元素，从该元素做指定范围的查找

注：通配符是指查找页面所有的标签，不到万不得已不推荐使用
```
### 3.选择器优先级
``` python
    行间样式   1000
    id选择器   100
    类选择器   10
    标签选择器 1
    
    根据选择器权重进行计算：优先级级越高，优先显示；权重一样，后者覆盖前者
```
### 4.元素分类
``` python
    (1)块元素：
        1) 独占一行
        2) 没有宽度时，默认撑满一行
        3) 支持所有css命令
    (2)内联元素：
        1) 一行上能显示多个同种类型的元素 
        2) 内联元素不支持宽高
        3) 不支持上下的margin
        4) 内容撑开宽度
        5) 代码换行被解析，解析成空格
    (3)内联块元素：
        1) 能让块元素在一行上显示多个
        2) 内联元素支持宽高
        3) 内联元素支持margin
        4) 一行上能显示多个同种类型的元素 
        5) 内容撑开宽高
        6) 代码换行被解析，被解析成空格
        7) 支持所有css命令
    
    元素区分：
        块元素有： body,div,hr,h1--h6,p,ul,ol,li,dl,dt,dd,form,option
        内联元素有：mark.strong,em,span,label,radio,checkbox
        内联块元素有：input,button,select,textarea
        特殊元素： img
```
### 5.display inline/block/inline-block
``` python
    display改变的是标签在显示时的特征，并不影响该标签的本质
```
### 6.编码规范
``` python
    1) 所有书写必须在英文半角状态下的小写
    2) id,class必须以字母开头
    3) 所有标签必须闭合
    4) html标签用tab键缩进
    5) 属性值必须带引号
    6) /*css注释*/ <!--html注释-->
    7) ul,li/ol,li/dl,dt,dd是拥有父子级关系的标签
    8) p,dt,h标签，里边不能嵌套块属性的标签
    9) a标签不能嵌套a
    10) 内联标签不能嵌套块标签
```
