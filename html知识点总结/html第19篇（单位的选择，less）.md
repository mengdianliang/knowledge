### 1. 单位的选择
``` python
    (1) 像素px:
        绝对单位，任何情况下，都是固定值；
        挑战失败：不能在不同的设备下保持统一布局结构
    (2) 百分比： %

    相对单位，会根据父级的宽高进行百分比计算
    挑战失败： 还没到展示时，就不知道高度如何设置了
    (3) 文字单位em
    相对单位，根据字体大小进行计算
    挑战失败：根据字体大小得出来的值还是固定的，并且每一个容器有可能字体大小不一样，从而导致需要重新计算
    (4) rem
    相对单位
    优势： 只受html字体大小的影响
```
### 2. 移动端注意事项：
``` python
    (1) 固定定位的处理
        绝对定位：虽然全浏览器兼容，但会导致页面滚动条卡顿
        解决方案：用js处理滚动
    (2) 所有单位全部都用rem;
    (3) img要设定宽或高
    (4) 背景图： 需要使用background-size设置背景图大小；
    当容器宽度与背景图宽度一致时，可以使用100%；
    当不一致时，使用rem值
```
